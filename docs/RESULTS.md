# Result Files

The `results/` folder contains compact summary files rather than raw logs or checkpoints.

## Tables

- `tables/vtab1k_aoft_style_compact.csv`: compact VTAB-1K table.
- `tables/vtab1k_group_summary.csv`: natural/specialized/structured mean summary.
- `tables/vtab1k_natural.csv`: VTAB natural tasks.
- `tables/vtab1k_specialized.csv`: VTAB specialized tasks.
- `tables/vtab1k_structured.csv`: VTAB structured tasks.
- `tables/fgvc_main_comparison.csv`: FGVC comparison table.

## Analysis

- `analysis/selector_ablation.csv`: fixed/random/support selector ablation summary.
- `analysis/vtab_paired_significance.csv`: paired comparison summary.
- `analysis/vtab_average_rank.csv`: average rank summary.
- `analysis/fgvc_paired_gaps.csv`: FGVC paired gap summary.
